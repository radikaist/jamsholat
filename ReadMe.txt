🌟 Fitur Unggulan
Real-time Prayer Schedule: Data otomatis diambil dari API Kemenag (MyQuran) yang akurat untuk seluruh wilayah Indonesia.

Dynamic Location: Ubah lokasi kota langsung melalui Panel Admin tanpa menyentuh baris kode.

Countdown Timer: Penghitung waktu mundur otomatis menuju jadwal sholat berikutnya.

Waktu Syuruq: Dilengkapi dengan tampilan waktu terbit matahari (Syuruq).

Notifikasi Adzan: Pemicu suara adzan otomatis yang dapat disesuaikan sumber audionya (.mp3).

Custom Running Text: Panel admin untuk mengubah pesan teks berjalan (pengumuman/ayat) secara real-time.

Zero Database: Menggunakan LocalStorage browser untuk menyimpan pengaturan, sehingga tidak memerlukan server atau database eksternal.

Full Screen Mode: Mendukung mode layar penuh untuk tampilan display profesional.

🛠️ Stack Teknologi
HTML5 & JavaScript (Vanilla): Inti logika aplikasi tanpa dependensi berat.

Tailwind CSS: Untuk desain UI yang modern, responsif, dan elegan (Dark Mode).

Kemenag API (MyQuran): Sebagai sumber data jadwal sholat yang terpercaya.

🚀 Cara Penggunaan
Aplikasi ini bersifat Portable (All-in-One File). Anda tidak perlu melakukan proses instalasi yang rumit:

Clone repositori ini atau unduh file JamSholat.html.

Buka file JamSholat.html menggunakan browser modern (Chrome/Edge/Firefox).

Untuk mengatur lokasi dan pesan, klik ikon ⚙️ Pengaturan di pojok kanan atas.

Penting: Klik satu kali di area mana saja pada layar saat aplikasi dibuka agar browser mengizinkan pemutaran suara adzan secara otomatis.

⚙️ Panel Admin
Akses panel admin tersedia langsung di dalam aplikasi untuk mengatur:

Pencarian dan pemilihan Kota/Kabupaten.

Isi pesan Running Text.

URL file Audio Adzan.

Password default: admin123 (Dapat diubah di dalam kode).
